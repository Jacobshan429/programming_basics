# 程式設計入門

## 課程介紹

### 目標讀者
程式設計初學者
### 目的 
了解什麼是程式語言、程式語言基本語法，包括常見資料類型(數字、字串)、變數、函式、控制流程、陣列、雜湊、物件
### 程式語言 
Ruby
### 環境
* 線上: https://repl.it/languages/ruby
* Mac: 使用內建的 Ruby 和編輯器 https://www.sublimetext.com/
* Non-Mac: 使用 https://c9.io/ 開一個 Custom 的 workspace
